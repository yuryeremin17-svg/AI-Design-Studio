Новый клиент. Входные данные: $ARGUMENTS

Шаг 1 — Структура:
1. Создай папку output/[имя-клиента]/
2. Создай папку assets/logos/[имя-клиента]/

Шаг 2 — Мини-бриф (максимум 2 вопроса):
- Стиль: luxury / modern / corporate / tech / minimal? (покажи варианты из CLAUDE.md)
- Что делаем: брендбук Старт / Бизнес / Премиум?

Шаг 3 — Дизайн-решения (предложи сам, не спрашивай):
- Подбери шрифтовую пару из CLAUDE.md по стилю
- Подбери палитру (3-4 цвета: primary, accent, secondary, light)
- Создай логотип (SVG: основной + light для тёмного фона + icon для аватарки)

Шаг 4 — Покажи решения клиенту:
- Палитра + шрифты + логотип — на одном экране
- Дождись одобрения или правок

Шаг 5 — Производство:
- Создай брендбук (HTML, дизайн-система v3: CSS-переменные, 8px grid, reveal-анимации, progress bar, copy HEX)
- Визитка встроена в брендбук (секция 09)
- Проверь через Playwright

Шаг 6 — Доставка:
- Запусти: node scripts/deliver.js [имя-клиента]
- ZIP готов в delivery/[имя-клиента]/

Шаг 7 — Запиши клиента в MEMORY.md
