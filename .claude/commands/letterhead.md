description: Создать фирменный бланк
---

Создай фирменный бланк A4 (HTML) для клиента.

Входные данные: $ARGUMENTS

Два варианта: стандартный + с водяным знаком.
Шапка: логотип + контакты + двойная линия.
Подвал: одинарная линия + копирайт.
Используй стиль из текущего брендбука клиента (если есть в output/).
Сохрани в output/[имя-клиента]/letterhead.html
