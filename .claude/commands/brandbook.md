Создай полный брендбук (HTML, один файл) для клиента.

1. Разбери входные данные (название, сфера, стиль, цвета, контакты)
2. Подбери шрифтовую пару из CLAUDE.md
3. Подбери палитру (или используй указанную клиентом)
4. Создай файл output/[имя-клиента]/brandbook.html со всеми 13 разделами
5. Логотип возьми из assets/logos/
6. Проверь визуально
7. Сообщи что готово и как открыть
