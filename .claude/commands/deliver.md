description: Собрать и отправить финал клиенту
---

Подготовь финальную доставку для клиента.

Входные данные: $ARGUMENTS (имя клиента, email)

1. Найди все файлы в output/[имя-клиента]/
2. Список готовых материалов
3. Напомни: "Откройте каждый HTML в Chrome → Ctrl+P → Сохранить как PDF"
4. Если есть MCP Gmail — предложи отправить на email
