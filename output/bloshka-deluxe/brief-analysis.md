# Bloshka DELUXE — Анализ брифа

## Шкалы клиента
- modern: 4/5 (современный, не устаревший)
- playful: 3/5 (умеренно игривый)
- luxury: 2/5 (НЕ холодный люкс, а тёплый премиум)
- feminine: 4/5 (женственный, мягкий)
- tone: friendly (дружелюбный)

## Слова бренда
Красота, вкус, эстетика, качество, надёжность

## Эмоции бренда
Восторг, доверие, желание вернуться

## Персона бренда
Интеллигентный, образованный, хорошо воспитанный, стильный, с утончённым вкусом

## Что нравится (admired_brands)
**Kaori (kaoristore.ru)** — нишевая косметика, НЕ винтаж
- Минималистичный дизайн, много воздуха
- Премиум без пафоса
- Формально-дружелюбный тон, забота о клиенте
- Цвета: чёрный, белый, песочный
- Шрифт: FuturaPT (современный sans)
- **Что взять:** подход "премиум без пафоса", чистоту дизайна, заботу о клиенте

## Конкуренты

### 3 Сороки (3soroki.ru)
- Скандинавский минимализм + винтаж
- Шрифты: Philosopher + Roboto
- Тон: дружеский, экспертный, ностальгический
- Ценовой сегмент: премиум-люкс (900-50000+ руб)
- Сильные: узнаваемая идентичность, Энциклопедия, блог
- Слабые: перегруженный каталог, сложная навигация

### Винтажная лавка / NeUstarelo (neustarelo.ru)
- Бордо #771e33 + белый
- Шрифты: Cormorant + Manrope (!)
- Тон: философский, эмоциональный, личный ("душа вещи", "замедлиться")
- Ценовой сегмент: премиум (3000-30000+ руб)
- Сильные: личная история, эмоциональная связь с товаром
- Слабые: мало характеристик товара, короткие отзывы

## Дифференциатор (из брифа)
"У них дороже и почти нет изделий царской и советской России"

## Что клиент НЕ хочет
- Чёрный цвет (avoid_colors)
- Холодный люкс (luxury=2/5)
- Пафос (tone=friendly)

## Нравящиеся логотипы
Шанель, Диор, Селин — классика, минимализм, элегантность

## Дизайн-решения на основе анализа

| Решение | Обоснование |
|---------|-------------|
| Cormorant Garamond + DM Sans | Cormorant подтверждён рынком (NeUstarelo), DM Sans — современнее Roboto (как у Kaori) |
| Dusty Plum #5C3A4E (не бордо) | Мягче, чем бордо конкурента NeUstarelo. Клиент просил "приглушённые глубокие" |
| Тёплый тон, не холодный | luxury=2/5, tone=friendly, admired=Kaori (премиум без пафоса) |
| Русский как основной язык | Клиент из СНГ, аудитория русскоязычная |
| "Каждая вещь — история" | Подход NeUstarelo работает, но мы теплее и доступнее |
| УТП: царская и советская Россия | Прямо из differentiator в брифе |
| Без чёрного | avoid_colors из брифа |
