- Доставка: HTML→PDF на старте, Figma к Фазе 3

## Уроки
- PDF нельзя редактировать программно без потери качества — всегда HTML
- Логотип вставлять из файла, не воспроизводить кодом
- Каждая правка должна быть точечной — не пересоздавать всю страницу
- Перед сдачей обязательна визуальная проверка
- Шрифты только Google Fonts — проприетарные шрифты не работают в HTML
- AI ускоряет производство x3-5, но коммуникация с клиентом — x1 (ноль экономии)
- Реальная экономия на полном цикле — x2
